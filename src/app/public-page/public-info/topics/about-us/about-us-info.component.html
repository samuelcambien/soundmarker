Hi, we are
Soundmarker!

Soundmarker is a simple tool to improve the communication
during the creative process of making music.

Share as you wish

Set your expiration time dependent on the project and choose if the recipient can download the tracks.

Stream anywhere

You and your recipients can stream your shared tracks on any device wherever you are.

Set the markers

You can add comments on any second in a track or select regions if you need to be more specific.

Smart search

Immediately find the comments you need while working on the creative side of a project.
